# Dual Enrollment: Web Page Development Summer 2026

**Live Demo:** https://aashigam.github.io/de-webdev-media-integration/

## Project Overview
This project was developed as part of my Dual Enrollment (DE) Web Page Development course (Module 4: Working with Media). The objective was to design a responsive, accessible webpage that integrates diverse digital media formats. The project showcases a clean layout of curated blue nature elements while demonstrating how different file types render on the web and how to structure layout flow around visual content.

## Key Technical Skills Demonstrated
* **CSS Layout Controls:** Implemented wrapping layouts using `float: left` and text-flow positioning.
* **Format-Specific Asset Management:** Used distinct media types including JPEG (for photograph gradients), PNG (for lossless graphics), and animated GIFs.
* **HTML5 Multimedia Embeds:** Programmed native `<audio>` and `<video>` tags with standard browser controls to play streaming ambient waves and background motion video.
* **Academic Context:** Built to meet rigorous collegiate grading criteria for media compression, asset attribution, and user accessibility.
